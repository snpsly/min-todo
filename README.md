# 基于Vue3技术栈的个人待办应用——min-todo

## 开源协议

 AGPL3 协议开源

可自用，额外允许二次开发后上架到想天浏览器应用市场

## 技术框架

项目由Vue3 + TS + Vite +Pina 搭建

此项目为想天轻应用，也可以单独运行在web上。

## 功能：

✅ 实现待办的添加

todo:实现清单

todo:实现想天浏览器小组件

todo:转化为想天浏览器轻应用

todo:上架到线上

线上demo:

可直接添加到书签使用
